# rustbucket

A collection of microservices and utilities

## Apps & Services

### **gatekeeper**
API Gateway with authentication, CORS handling, and service routing

### **next-rusty-tech**
Next.js frontend application with Spotify integration and user authentication

### **operator**
SMS-based monitoring service for infrastructure health checks via Twilio webhooks

### **spotify**
Flask REST API for Spotify OAuth integration and music search features

### **squirrel**
FastAPI web scraper for real-time NFL player/team data with caching and rate limiting

### **nothing**
PDF bank statement parser and transaction analyzer for spending trend tracking

### **hue-dashboard**
Flask web interface for Philips Hue smart light control

### **packages/animations**
Reusable CSS/JS animation library for web projects
